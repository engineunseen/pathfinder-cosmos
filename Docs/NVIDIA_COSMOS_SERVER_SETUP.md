Для полноценной работы Pathfinder с ИИ-мостом Cosmos, на вашем компьютере **одновременно** должны быть запущены два процесса в разных терминалах:

1. **Основное приложение**:
   ```bash

   ```
2. **Прокси-сервер (Bridge)**:
   ```bash

   ```
*Без запущенного `proxy.js` игра не сможет "достучаться" до сервера Brev.dev из-за политик безопасности браузера (CORS).*

## 📡 Настройка прокси-моста (Stealth Bridge v3.3.36)

Для стабильной работы без ошибок CORS и "BadRequest" (400) мы используем локальный прокси `proxy.js`.

### 1. Настройка сетевого доступа (Brev.dev & VS Code)
1.1 В консоли Brev.dev найдите настройки порта **8000**.
1.2 **ОТКЛЮЧИТЕ Cloudflare Access**: установите режим **"Public"**.
1.3 Если вы работаете в VS Code Remote/Codespaces, внизу во вкладке **Ports** найдите порт **8001** (наш прокси). Кликните правой кнопкой и установите **Port Visibility -> Public**. 

*ВАЖНО: Эти ссылки должны быть публичными в обоих местах, чтобы браузер мог достучаться до API через прокси без блокировок.*

### 2. Принцип "Secret Token"
- В целях безопасности (так как порт открыт) мы используем вашу **ссылку Brev (https://...) как пароль**.
- Игра отправляет эту ссылку в заголовке `x-target-url`. Прокси принимает её и перенаправляет запрос на реальный сервер.
- Таким образом, никто не сможет воспользоваться вашим сервером, не зная его уникального ID.

### 3. Авто-оптимизация (v3.3.31+)
- Прокси автоматически сжимает картинки до **512px JPEG** перед отправкой на сервер. Это устраняет ошибки `BadRequest` и снижает нагрузку на CPU/каналы связи в 10 раз.
