# Спецификация Режима "Архитектор": Стратегическое Планирование (v0.9.6)

Этот документ описывает двухуровневую систему ИИ-анализа в проекте Unseen Pathfinder.

## 1. Архитектура "Глубокого Рассуждения"
Миссия больше не полагается на один запрос. Мы разделили интеллект на две критические роли:

1.  **AI ARCHITECT (Стратег)**:
    *   **Вход**: Хайтмап 257x257 + Координаты.
    *   **Задача**: Глубокий технический анализ топологии. Идентификация кратеров, базальтовых гряд и валунов.
    *   **Выход**: JSON с массивом вейпоинтов и подробным многостраничным отчетом (`reasoning`).
    *   **Философия**: "Нет прямым линиям". Обход препятствий с учетом градиента высот.

2.  **PHYSICAL SPECIALIST (Научный сотрудник)**:
    *   **Задача**: Анализ физических свойств среды (реголит, трение, гравитация).
    *   **Выход**: Технически плотные научные отчеты и предупреждения.
- **Waypoint Resolution**: 3-уровневый выбор (7 / 15 / 25 точек).
- **AiTrail**: Визуальный неоновый шлейф, отображающий реальный путь ИИ. Исчезает при ручном управлении.
- **Cognitive Sensory Layer**: Определение метрик sCVaR (риск) и SMaR (устойчивость) непосредственно в промпте.

## 3. Честность и Прозрачность
*   **Никакой Эвристики**: Мы удалили все "запасные" алгоритмы. Если ИИ недоступен — система выдает `FATAL ERROR`. Миссия либо выполняется ИИ, либо не выполняется вовсе.
*   **Терминал**: Все рассуждения ИИ (Architect Analysis) выводятся в терминал в сыром, нефильтрованном виде. Это позволяет оператору видеть "ход мыслей" машины.

## 4. Синхронизация: NVIDIA Cosmos
Проект подготовлен к приему данных от **NVIDIA Cosmos Reasoning**. Формат обмена данными (PNG 257x257 -> JSON JSON) полностью унифицирован.
