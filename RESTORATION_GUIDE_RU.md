# Инструкция по восстановлению проекта

1.  **Клонирование**: `git clone https://github.com/engineunseen/pathfinder.git .`
2.  **Установка**: `npm install`
3.  **Запуск**: `npm run dev`
4.  **Логика**: См. `./PROJECT_ARCHITECTURE.md` и `./src/aiNavigator.js`.

**No push protocol**: Только PULL из основного репозитория.
